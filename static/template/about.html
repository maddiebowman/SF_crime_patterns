About This Project

This project is a web application designed to visualize various data points on an interactive map and other interactive visuals. Users can filter data by categories and navigate between different views including a heatmap and bar chart.

The map is built using Leaflet.js. Data clustering is handled by the Leaflet MarkerCluster plugin, and D3.js is used for data manipulation and visualization.

For more information, please visit our GitHub repository.

API Endpoints

Endpoint Description Usage
/data Get full data from the database. This endpoint retrieves all records from the incidents_full collection. This endpoint is used for the crime projection.
/reduced_data Get reduced sample data. This endpoint retrieves all records from the incidents collection, which contains a smaller, sample dataset. This endpoint is used for the Leaflet map and ApexCharts heatmap.
/aggregated_data Get aggregated data. This endpoint aggregates data by year, month, and category, providing counts of incidents based on the full dataset. This end point is used for the bubble animation visulization.
/selected Get selected data. This endpoint retrieves specific fields (Latitude, Longitude, Incident Datetime, Incident Description, Analysis Neighborhood, Incident Category, Incident Year) from the incidents collection. Further narrowed down the fields from reduced_data.
/selected_<type> Get data for a specific category. Replace <type> with the category name to retrieve records that match a specific incident category.
